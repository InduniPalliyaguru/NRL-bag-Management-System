A robust and centralized Java-based desktop application developed as a **Semester 1 Final Project** to digitize, automate, and streamline the daily operations of a small-scale bag manufacturing business.

🧵 **The Problem:** The business previously relied on a fully manual, notebook-based workflow, which was highly prone to errors, data loss, and tracking inefficiencies.
🚀 **The Solution:** NRL Bag Management System replaces the manual system with a standalone digital application, bringing accuracy and efficiency into managing customers, stock, orders, and finances.

---

### 💻 Key Features & Core Modules

* **👤 Customer Management:** Complete digital registration, updating, and historical record tracking of all active clients.
* **📦 Bag Order Tracking:** End-to-end monitoring of production pipelines and customer bag orders from initiation to delivery.
* **📊 Material Stock Management:** Live inventory tracking of raw materials to prevent shortages and optimize supply planning.
* **💰 Financial & Payment Handling:** Accurate ledger tracking for processing advance payments, outstanding balances, and final settlements.
* **🔍 Advanced Data Retrieval:** Structured search and indexing mechanisms to query information instantly across the database.
* **📈 Report Generation:** Basic, clean business reporting tools to extract essential metrics for data-driven decision-making.

---

### 🛠️ Tech Stack & Architecture

* **Language:** Java
* **UI Framework:** JavaFX (for building a responsive, intuitive desktop user interface)
* **Database:** MySQL (Relational database management for persistent data storage)
* **IDE:** IntelliJ IDEA
* **Architecture Pattern:** Organized with clean code principles separation to isolate UI controller layers from underlying business logic and database queries.
